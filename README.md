# 🥘 Yummy - Food Recipes Application

**Yummy** is a dynamic and responsive web application built to explore various food recipes from around the world. The application allows users to search for meals, filter by categories, and view detailed cooking instructions and ingredients.

## ✨ Features
* **Global Search:** Find recipes by name or first letter.
* **Filter by Category:** Browse through various food categories like Beef, Chicken, Pasta, and more.
* **Area-Based Recipes:** Explore cuisines based on their geographic origin (Egyptian, Italian, American, etc.).
* **Detailed Instructions:** Comprehensive cooking steps with a list of required ingredients and measures.
* **Responsive Sidebar:** Easy navigation through search, categories, ingredients, and contact sections.

## 🚀 Technologies Used
* **Frontend Framework:** Angular
* **Styling:** Bootstrap 5 & Tailwind CSS for a modern, responsive UI
* **Scripting:** JavaScript (ES6+) & jQuery
* **API:** Integration with a Meals DB to fetch real-time data.
* **Version Control:** Git & GitHub
